# 機能詳細

## 出品機能

### 出品タイプ
- **Webサイト** 🌐: ブログ、ECサイト、メディアサイトなど
- **アプリ** 📱: iOS/Androidアプリ
- **SaaS** ☁️: サブスクリプション型サービス

### 価格プラン設定

出品時に複数の価格プランを設定できます。

#### プラン例

**基本プラン**
- ソースコードのみ
- 価格: ¥100,000

**スタンダードプラン**
- ソースコード
- 会員データ（ユーザー情報、メールアドレスなど）
- 価格: ¥150,000

**プレミアムプラン**
- ソースコード
- 会員データ
- 設置サポート（サーバー設定、初期セットアップ）
- 価格: ¥200,000

#### プランに含められる内容

- **会員データ**: ユーザー情報、登録データ
- **ソースコード**: プログラムのソースファイル
- **設置サポート**: サーバー設置、初期設定のサポート

### KPI指標

#### Webサイト向け
- **月間PV**: ページビュー数
- **月間UU**: ユニークユーザー数
- **月間売上**: 広告収入、アフィリエイト収入など
- **月間利益**: 経費を差し引いた純利益

#### アプリ向け
- **登録ユーザー数**: 累計登録者数
- **DAU**: デイリーアクティブユーザー
- **MAU**: マンスリーアクティブユーザー
- **累計ダウンロード数**: App Store/Google Playでのダウンロード数
- **月間売上**: アプリ内課金、広告収入など
- **月間利益**: 経費を差し引いた純利益

#### SaaS向け
- 上記すべての指標を登録可能

## 決済・取引機能

### エスクロー機能

安全な取引のため、エスクロー（第三者預託）機能を実装しています。

#### 取引フロー

1. **購入者が決済** → 資金はプラットフォームが一時保管
2. **販売者が商品を引き渡し** → ソースコード、会員データなどを提供
3. **購入者が確認** → 商品を受け取ったことを確認
4. **支払い解放** → 販売者に資金が送金される

#### 手数料

- プラットフォーム手数料: **10%**
- 例: ¥100,000の取引 → 販売者受取額 ¥90,000

### 決済方法

Stripeを使用した安全な決済：
- クレジットカード
- デビットカード
- その他Stripeがサポートする決済方法

## 会員機能

### 会員登録（無料）

- メールアドレスとパスワードで登録
- 登録後すぐに購入者・販売者として利用可能

### 権限

すべての会員が以下の機能を利用可能：
- 出品の閲覧・購入
- 自分のサイト・アプリの出品
- 取引履歴の確認

### 認証不要で利用できる機能

- 出品一覧の閲覧
- 各出品の基本情報（タイトル、価格、種類）の確認

### 認証が必要な機能

- 出品の詳細情報の閲覧
- 購入手続き
- 出品の作成・編集
- 取引履歴の確認
